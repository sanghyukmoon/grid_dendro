References and Historical Remarks#

  • Gong and Ostriker [2011]: GRID

  • Mao et al. [2020]: FISO

Gong and Ostriker [2011] developed and introduced a core finding method called GRID (GRavitatianal IDentification), which utilize isocontours of gravitational potential to provide physically motivated definition of bound cores. Mao et al. [2020] extended this algorithm to identify hierarchical structures of the interstellar medium using dendrogram, as well as significantly improve the efficiency of the algorithm. The first author of Mao et al. [2020] wrote a python pacakge FISO (fast isocontours) to implement this algorithm.

grid_dendro is an implementation of an algorithm presented in Mao et al. [2020], written by refactoring the original FISO code base to enhance readability and maintainability.
